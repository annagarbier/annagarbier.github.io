What is this?

Naive Thesaurus exposes the importance of context to the meaning of words.

When "Synonym-ize" is pressed, each word in the text box is replaced with a completely random synonym from the Merriam-Webster Thesaurus (if available).

How does it work?

This project uses the Merriam Webster Thesaurus API to do on-the-fly synonym lookup and replacements. Replacements may be recursive ad infinitum.
