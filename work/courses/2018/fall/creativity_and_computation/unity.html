Unity HW 01

Using our introductory class material and Unity documentation, I created a set of floating cubes that spin via input from the Inspector. For every full rotation that a cube makes, the C# script also changes the cube's material color.

One cube object hovers above a plane object. Both are given material colors via the Unity GUI during initial setup. The cube object is then connected to a simple C# script called Spin. In its first iteration, the script causes the cube to spin at a default speed of 10 degrees per second. This speed is held in a variable, so we can use the Inspector to change the speed.

In its second iteration, the Spin script allows us to change the cube's spinning speed in the Inspector using a slider that's constrained to 0 - 500 degrees per second. The script also changes the color of the cube's material after every full rotation of the cube.

The same Spin script is now added to eight additional cube objects. Each cube is now independently configurable in the Inspector.
