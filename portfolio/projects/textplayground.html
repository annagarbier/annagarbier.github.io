Text Playground

The Text Playground is a web-based interface for building text manipulation programs. By using visual nodes, the playground separates the ability to think computationally about language from the ability to write code. This project is in progress.

Credits
The Text Playground uses Vitaliy Stoliarov's rete.js library. The current demo version was built as an indepenent study at Parsons School of Design with advising from Justin Bakse.
